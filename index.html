Troika! System JSON

JSON data structure for the Troika! tabletop RPG system

JSON Schemas

troika-system.schema.json - Complete system schema background.schema.json - Character background schema character.schema.json - Player character schema enemy.schema.json - Enemy/creature schema item.schema.json - Equipment and items schema skill.schema.json - Character skills schema spell.schema.json - Magic spells schema table.schema.json - Game tables schema

Data Objects

troika-system-data.json - Main data aggregation file backgrounds/ - Character backgrounds (36 files) enemies/ - Bestiary entries (36 files) items/ - Equipment and items (50 files) skills/ - Character skills (22 files) spells/ - Magic spells (74 files) tables/ - Game tables (5 files)
